## Install

Install ecommerce com npm on main page

```bash
  cd mech_service
  npm install
  
```

## Deploy

To deploy this project run in the main folder:

```bash
  npm run dev
```

now front end is running. 

```bash
  npm run fake-backend
```

to run mock backend
